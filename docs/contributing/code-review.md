# Code Review

Review exists to protect the long-term health of NOVUS OS.

## Review Priorities

- Correct folder ownership.
- Clear module boundaries.
- Security and privacy concerns.
- Documentation updates.
- Testability.
- Maintainability.
- Dependency risk.

## Foundation Phase Review

During foundation mode, reviews should confirm that no product code, framework initialization or dependencies are introduced without the required decision process.
