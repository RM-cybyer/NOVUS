Define the handoff format Codex must leave after repository guardian work.

```text

-

- 

Affected files:
- 

Affected documentation:
-

-
```

## Handoff Rules

- Handoffs must be specific enough for Fable, Claude, QA or a future agent to continue immediately.
- If GitHub was not updated, state that clearly.
- If repository verification did not run, explain why.
