# Claude Context

Claude is expected to own vision, roadmap, product strategy, AI philosophy and high-level engineering direction.

## Boundary

Repository maintenance work should not override Claude-owned strategic decisions. If a repository change requires strategic input, create a handoff using `.ai/shared/agent-handoff-template.md`.

## Expected Inputs From Claude

- Product vision updates
- Roadmap decisions
- Feature priorities
- Strategic architecture direction
- AI behavior philosophy
